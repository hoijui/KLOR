# KLOR SAEGEWERK 3DP BLE CASE

You can choose a case with or without support for the SplitKB tenting puck.\
The 3DP cases I use were printed in PA-12-HP Nylon (MJF) by [JLCPCB](https://jlcpcb.com/).
